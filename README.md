# QLoF Primer

**Quantum Laws of Form — First Principles for Everyone**

One simple idea — *drawing a line* — unifies math, physics, chemistry, biology, psychology, engineering, and daily life. No jargon. No prerequisites. For anyone who's ever been curious about how things work.

---

## What This Is

Based on George Spencer-Brown's *Laws of Form* (1969), this primer extracts the core insight — that everything begins with the act of drawing a distinction — and shows how it applies across every discipline. It's a lens, not a textbook. Once you learn to see the world as patterns of distinctions, you can't unsee it.
